# OLAP与OLTP

数据处理分类

1. 联机事务处理OLTP(on-line transaction processing):传统关系数据库的主要应用,主要是基本的,日常的事务处理
2. 联机分析处理OLAP(On-Line Analytical Processing): 数据仓库系统的主要应用,支持复杂的分析操作, 侧重决策支持,并且提供直观易懂的查询处理结果
